# Jet Cube Turbofan MVP

CMAPSS Turbofan Engine Degradation Simulation (NASA) üzerinden bakım/arıza öngörüsü (RUL) ve anomali tespiti için MVP çalışma alanı.

## Klasörler
- `data/raw/`: Ham veri
- `data/processed/`: İşlenmiş veri
- `data/outputs/`: Çıktılar (modeller, grafikler, raporlar)
- `notebooks/`: Analiz notebook'ları
- `docs/`: Dokümantasyon
- `demo/`: Demo materyalleri
